## 🎓 How to Use the Design System

### For Colors
```css
.element {
  color: var(--text-primary);        /* Dark text */
  background: var(--color-primary);  /* Primary color */
  border: 1px solid var(--color-primary-rgb), 0.15); /* Semi-transparent */
}
```

### For Spacing
```css
.component {
  padding: var(--space-lg);     /* 16px */
  margin: var(--space-xl);      /* 24px */
  gap: var(--space-md);         /* 12px */
}
```

### For Typography
```css
h1 {
  font-family: var(--font-secondary);  /* Tinos */
  font-size: clamp(1.5rem, 5vw, 3rem);
  font-weight: 700;
}

p {
  font-family: var(--font-primary);    /* Poppins */
  font-size: 1rem;
  line-height: 1.6;
}
```

### For Shadows
```css
.card {
  box-shadow: var(--shadow-md);  /* Standard card */
}

.card:hover {
  box-shadow: var(--shadow-lg);  /* Elevated on hover */
}
```

### For Transitions
```css
button {
  transition: all var(--transition-normal);  /* 300ms */
}

input:focus {
  transition: color var(--transition-fast);  /* 150ms */
}
```
